# IMDb Analytics

<b>Author:</b> Nilrudra Mukhopadhyay<br>
<b>Project Type:</b> Data warehousing and Predictive Analytics project - independently built<br>

## Project Overview

Business question:

- Based on given information on a new movie such as ratings, genre, and crew. Can we predict how much a movie will make at the box office during Opening weekend, domestic gross, and/or international gross profits?
- Can we also predict what it's ratings would look?

Tech Stack:

- Python
- Scikit-learn, Tkinter
- Microsoft SQL express server
- Microsoft SQL sever management studio (SSMS)
- Visual studio code
- Git bash/Github

## Data Organization

Data Sources:

- [OMDb API open IMDb database, IMDb ID based](https://www.omdbapi.com/)
- [Free IMDb API, IMDb ID based](https://imdbapi.dev/)
- [Publicly available IMDb dataset, ID fragment](/data/imdb_movie_ids.txt)
